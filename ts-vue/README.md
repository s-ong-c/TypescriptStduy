# ts-vue

## Project setup
1. 준비
타입스크립트 설치
```console
$ npm install -g typescript
```
vue cli 설치
```console
$ npm install -g @vue/cli 
 $ yarn global add @vue/cli

```
vue 프로젝트 생성
```console
$ vue create <project name> 
```
